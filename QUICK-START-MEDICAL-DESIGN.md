
### Option 2: Make Permanent
If you want to make these the default files:

```bash

cd frontend/src

# Backup originals

mv layouts/public/PublicLayout.js layouts/public/PublicLayout-original.js

# Rename medical versions to default
mv pages/home/Home-Medical.js pages/home/Home.js

# Update App.js imports (change back to standard names)
```
